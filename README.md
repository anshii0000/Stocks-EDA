# Stocks-EDA
📈 Stock Market Exploratory Data Analysis (EDA)  This project performs a comprehensive Exploratory Data Analysis (EDA) on historical stock market data to uncover trends, patterns, and insights from stock prices and trading volumes. The analysis involves data cleaning, statistical exploration, feature engineering, and visualizations to better understand stock performance over time.  
Using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn, the project analyzes key financial metrics including opening, closing, high, low, adjusted closing prices, and trading volume. It also derives additional features such as daily returns, price volatility, and moving averages to provide deeper insights into market behavior.  
🔍 Key Objectives 
Clean and preprocess stock market data. Perform exploratory data analysis using descriptive statistics and visualizations. Identify trends, outliers, and correlations among stock features.
Analyze daily price movements, trading volumes, and stock volatility.
Generate meaningful business insights to support data-driven investment decisions.
📊 Key Analyses Data quality assessment (missing values and duplicates)
Descriptive statistical analysis Distribution of stock prices and trading volumes 
Correlation analysis using a heatmap Time-series analysis of stock prices 
Daily return and volatility analysis 
Moving average trend analysis Company-wise performance comparison
